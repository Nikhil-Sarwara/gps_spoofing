### Abstract (~250–300 words)
| Element | Content |
|---|---|
| Problem | GPS spoofing poses a critical security threat to UAVs; existing solutions require specialised hardware |
| Approach | Software-only detection using GPS-IMU consistency cross-validation + ML classifiers (RF, 1D CNN) |
| Method | PX4 SITL simulation, MAVLink telemetry at 10Hz, 8 heuristic labelling rules, 30-sample sliding windows |
| Key Results | RF: 100% test accuracy; CNN: perfect recall but 100% FPR on normal samples ("CNN Paradox") |
| Contribution | Physics-based discriminator that distinguishes spoofing from environmental noise without hardware augmentation |
| Conclusion | Real-time software-only spoofing detection is feasible; limitations include 3s latency and simulation-only validation |
